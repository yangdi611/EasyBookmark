# 国内优秀的前端CDN
CDN要求速度快，项目多，版本更新快，国内BootCDN.cn我觉得是最好的，没有之一，什么新浪，百度，360的都是给内部用的。

最近发现的css.net也不错啊，还反代了Google字体库，够快，很方便。

    www.bootcdn.cn

    https://css.net/
    https://cdn.css.net/libs/

    https://cdnjs.com/

    https://www.jsdelivr.com/
